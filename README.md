# SIDIAM 2

SIDIAM é uma plataforma de diagnóstico em matemática para apoiar professores na identificação de dificuldades de aprendizagem, acompanhamento de turmas e geração de insights pedagógicos.

## Objetivo

Iniciar o projeto SIDIAM com foco no MVP definido em `docs/MVP-001-visao-e-requisitos.md` e na arquitetura técnica descrita em `docs/ADR-001-especificacao-tecnica.md`.

## Estrutura inicial

- `docs/` — documentação do MVP, ADRs e decisões arquiteturais
- `src/` — código-fonte do aplicativo
- `PROJECT-START.md` — plano inicial de tarefas e decisões de tecnologia

## Próximos passos

1. Escolher stack tecnológica (ex.: Node.js + React, Django + React, FastAPI + Vue, etc.)
2. Definir o modelo de dados inicial com base nas ADRs
3. Criar um protótipo de API / backend mínimo
4. Criar uma interface inicial para cadastro de professores, turmas e alunos

## Uso

A partir daqui, podemos:

- definir a primeira API REST
- criar a modelagem de dados para `users`, `schools`, `classes`, `students` e `assessments`
- construir o primeiro fluxo de cadastro de turma e registro pedagógico
